# SpaScope

**A Python package for multi-scale spatial structure analysis from spatial single-cell data**

</div>

SpaScope is a Python package for multi-scale spatial structure analysis from spatial single-cell data. It provides a unified workflow for graph-based feature extraction across candidate spatial scales, representative scale identification, spatial structure clustering, diversity analysis, raster-based landscape analysis, patch visualization, and contact score quantification.
